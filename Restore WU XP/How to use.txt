
********* How to USE *********

(1) Press on "Windows Update Installer". System will restart twice.
(2) Start Server from desktop --> 'Start server' file.
(3) Start IE8. Tools MENU. WU. Let him check for updates. wait 5 minutes.

-- Not work ?? read here. --

(4) Restart Computer.
(5) Press on "Windows Update Installer". System will restart twice.
(6) Start Server from desktop --> 'Start server' file.
(7) Start IE8. Tools MENU. WU. Let him check again.
  
********* Important Notes *********

- If you have Activation problem,
  Convert to Volume License. Try again.

- Windows Update server requires SHA-256, not supported by older versions (RTM/SP1/SP2).
  Windows 2000 (even with SP4) also will NOT work.
  
- Sometimes wuaueng.dll will not patch successfully. If this happens, go into the x86 folder,
  and copy wuaueng.dll to C:\Windows\System32 and C:\Windows\system32\dllcache. Click cancel
  on the warnings.
